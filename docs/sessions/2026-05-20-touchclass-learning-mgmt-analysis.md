# 2026-05-20 TouchClass 수강관리 화면 분석 — newcourse 참고용

원본: `reference/admin/learning/수강-관리-{요약,수강생,수강생등록,동영상,퀴즈,시험,설문,PDF,의견}.png` (9장)

## 전체 레이아웃 (chrome-on)

```
[글로벌 좌측 nav]              [노란 탑바]
[코스 메타 패널 좌측]    [수강관리 탭 영역 (오른쪽)]
                        ↓
                    [요약·수강·동영상·퀴즈·시험·설문·의견·PDF·라이브]
                    [우상단 수강생 등록 버튼 (항상 노출)]
```

### 좌측 코스 메타 패널 (모든 탭 공통)
- 코스 표지 이미지 (세로 카드)
- 코스명 (긴 텍스트 truncate)
- **학습 완료 조건** 표시
- **수강 기간** 표시 (운영중 칩) + "종료하기" 버튼
- **코스 QR 코드** (png·svg 다운로드 / 링크 복사)
- **분류**: 직무 역량
- 하단 액션: 코스 정보 / 푸시 메시지 / 공지사항 / 팝업

### 우상단 액션
- `[ + 수강생 등록 ]` 버튼 (인디고, 모든 탭에서 항상 노출)

## 탭 구조 (9개)

### 1. 요약 (진입 화면)
4개 통계 카드:
- **코스 정보**: 총 인원 N명 · 학습 인원수 N · 평균 진도율 % · 평균 학습 시간
- **코스 시험 정보**: 문항수 · 평균 정답률 · 만점수 · 주관식 · OX 문제
- **코스 퀴즈 정보**: 문항수 · 정답률 등
- **코스 설문 정보**: 문항수 · 평균 응답률 · 미응답 · 주관식 · OX 응답
- **그룹별 학습량** 섹션 (펼침)

### 2. 수강 (수강생 목록)
- 상단: 필터(전체 회원/정회원) + 정렬 + 다운로드
- 우측 액션: "전체 수강기간 변경" 등
- 테이블 컬럼: 체크박스 · 회원 사진 · 아이디 · 이름 · 소속(다중 태그) · **수강기간** · **진도율 %** · **학습시간** · 이수(✓)
- 페이지네이션 (1-126명)

### 3. 수강생 등록 (모달)
4 탭:
- **회원**: 전체 회원 OR 선택한 회원 (+ 회원 선택 버튼)
- **그룹**: 그룹 선택
- **대량 등록**: CSV 업로드 (아이디 list)
- **아이디 입력**: 직접 아이디 입력

### 4. 동영상
- 영상별 열 그룹 (영상 1, 영상 2 ...)
- 각 영상마다 **재생률 %** 표시
- 우상단: "동영상 학습조건 미충족" 빨간 인디케이터 (N분 이상 미시청)
- 항목별 ⓘ "조건 미달성 N분 (~분 ~~)" 안내

### 5. 퀴즈
- 컬럼: 회원·소속 · **응시**(녹색 ✓) · **정답** · **오답** · 답안 초기화 액션
- 우상단: 다운로드

### 6. 시험
- 컬럼: 회원·소속 · **점수** · **응시** · **정답** · **오답** · 답안 초기화
- 우상단: 다운로드

### 7. 설문
- 컬럼: 회원·소속 · **응답** · **미응답** · 답안 초기화
- 우상단: 다운로드

### 8. PDF
- 컬럼: 회원·소속 · 파일별(`Playing_xxx.pdf`) **조회수 / 다운로드 횟수**
- 우상단: 다운로드

### 9. 의견
- 상단: 항목 추가/필터 셀렉터
- 의견 N개 카운트 + 다운로드
- 빈 상태: "등록된 내용이 없습니다." (회색 아이콘 + 텍스트)

## 패턴 정리

| 요소 | 패턴 |
|---|---|
| 진입 | 요약 탭 (전체 지표 한눈에) |
| 수강생 추가 | 우상단 고정 버튼 → 4가지 등록 방식 모달 |
| 통계 표시 | 카드형 KPI (인원·진도율·시간·정답률 등) |
| 항목별 결과 | 동영상/퀴즈/시험/설문/PDF 각각 별도 탭, 동일 테이블 형태 |
| 의견 | 항목 무관 통합 탭 (모든 항목 의견 모음) |
| 일괄 액션 | "답안 초기화", "전체 수강기간 변경" 등 |
| 다운로드 | 각 탭 우상단 CSV 다운로드 |

## newcourse 적용 시 매핑 가이드

### 직접 매핑 (그대로 차용)
- 좌측 코스 메타 패널 (표지·QR·기간·종료하기·하단 액션)
- 우상단 "수강생 등록" 모달 (4가지 등록 방식)
- 요약 탭 KPI 카드 구조
- 수강 탭 컬럼: 회원·아이디·소속·수강기간·진도율·학습시간·이수
- 답안 초기화 / 전체 수강기간 변경 / 다운로드 패턴

### 적응 매핑 (newcourse 데이터 모델 반영)
- 동영상 탭 = newcourse 영상 항목별 시청률 (`progress >= 90`)
- 퀴즈 탭 → newcourse는 퀴즈/시험 분리 불명확 → 통합 검토
- 시험 탭 = newcourse 시험 항목 (`exam.cond.passScore` 기준 합격/불합격)
- 설문 탭 = newcourse 설문 항목 (`survey` 제출 여부)
- PDF 탭 = newcourse 자료(pdf) 항목 — **수료 무관이므로 통계만 (열람·다운로드)**
- 의견 탭 = `community.enabled` 항목의 통합 의견 (모든 항목 의견 합치기)

### Phase 1 스코프
필수:
- 요약 (KPI)
- 수강 (수강생 목록)
- 수강생 등록 모달

선택 (항목 유형 있을 때만):
- 동영상 / 시험 / 설문 / 자료(PDF) — 항목 존재 시 탭 노출
- 의견 — 의견 ON 항목 있을 때만 노출

### Phase 2+
- 그룹별 학습량 통계
- 답안 초기화 / 일괄 액션
- 학습조건 미충족 알림

## chrome-off vs chrome-on 정책

- newcourse 빌더 = **chrome-off** (이미 적용, [[project_newcourse_builder_chrome_off]])
- **수강관리는 chrome-on** — list/management 페이지이므로 글로벌 nav·탑바 유지 ([[project_newcourse_builder_chrome_off]] 정책 일치)

## 파일 위치 제안

- `admin/online-course-view.html` 또는 `admin/online-course-mgmt.html` 신규 (현재 빈 링크는 `online-course-view.html?id=…` 패턴)
- offline은 `admin/offline-course-view.html` 이미 존재

## 후속 작업 (사용자 OK 받은 후)

1. 좌측 코스 메타 패널 컴포넌트 신규
2. 8개 탭 라우팅·상태 관리
3. 요약 탭 KPI 카드 (newcourse stepValidation·progress 데이터 활용)
4. 수강 탭 테이블 + 등록 모달
5. 동영상·시험·설문·자료 탭 (항목 유형별)
6. 의견 탭 (항목별 community 의견 통합)
