# 2026-05-19 디디쌤 vs newcourse — 필수(수료) 옵션 on/off 비교

질문: 디디쌤도 필수(수료) 옵션을 on/off 할 수 있나?

답: **있음. 디디쌤은 2단(항목+코스) on/off, newcourse는 1단(항목)만 구현.**

## 디디쌤 (가이드 원문 기준)

### ① 항목 단위 수료기준 포함
- "커리큘럼을 구성하고 있는 항목 중에서 수료기준에 포함시킬 항목을 선택"
- 별도 "수료 기준 구성" 페이지에서 항목 선택 + 비율 드래그를 한 번에 수행

### ② 코스 단위 수료증 발급 on/off
- "수료 기준을 일정 비율 이상 충족 시, 수료증이 자동으로 발급되도록 설정이 가능"
- 수료증 자체를 끄거나, 양식(A/B) 선택 가능

## newcourse 현재 (online-builder-v1.html)

### ① 항목 단위 필수/선택 토글 (구현됨)
- `chapters[].items[].isRequired` (line ~2862)
- 항목 카드 인라인 토글 + 톱니바퀴로 이수조건 진입
- 사이드바 헤더에 "커리큘럼 N · 항목 N · 필수 N" 카운터 노출

### ② 코스 단위 수료증 발급 on/off — 미구현
- 빌더에 노출 안 됨
- 수료증 양식 선택 UI 없음
- 수료기준 비율 가중치 설정 없음

## 입력 방식 차이

| 항목 | 디디쌤 | newcourse |
|---|---|---|
| 항목 수료기준 포함 | 별도 페이지에서 일괄 선택 | 항목 카드 인라인 토글 |
| 비율 가중치 | 드래그 슬라이더로 시각 설정 | 미구현 (충족/미충족 이진) |
| 수료증 발급 | 코스 단위 토글 + 양식 선택 | 미구현 |

## 갭 (newcourse 미구현 분)

1. **코스 단위 "수료증 발급 사용" on/off** — Phase 1 검토 필요
2. **수료기준 비율 가중치 슬라이더** — Phase 2 (이미 별도 안건)
3. **수료증 양식 선택** — Phase 2 이상

## 적용 메모

- 빠른 토글: newcourse 우위 (인라인)
- 한눈 비율 설정: 디디쌤 우위 (별도 페이지)
- 둘 다 항목 단위 on/off 개념은 공유. 디디쌤은 위에 코스 단위 한 층이 더 있음.
