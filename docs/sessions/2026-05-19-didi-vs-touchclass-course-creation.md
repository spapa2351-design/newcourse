## 정정 사항

- 디디쌤 가이드에 **"온라인 클래스 만들기"라는 명칭은 없음**.
- Chapter 3 클래스 = **기간/회차/구독/과정** 4종.
- 온라인 모드는 **기간 클래스**(단일 클래스 + 동영상/라이브/시험/과제/설문)에 포함.
- 회차 클래스(오프라인)·출결패드는 이 보고에서 제외. 오프라인 검토 때 별도 정리.

## 디디쌤 - 온라인 강좌(기간 클래스) 등록 플로우

1. 클래스명 (최대 50자)
2. 수강 기간 — **수강 시작 후 N일(30/60/90/직접)** OR **날짜 지정**
3. 교육 목표 및 주요 내용 (텍스트·이미지·표)
4. 썸네일 (업로드 OR 기본)
5. 강사 소개 (다중 강사)
6. 커리큘럼 — **단일 클래스** OR **과정 클래스(기존 클래스 묶음)**
   - 구성 요소 5종: 동영상·라이브·시험·과제·설문
   - **라이브러리에서 가져오기** (이미 업로드된 자산 재사용)
7. 수료기준 + 수료증 — **드래그로 비율 설정** (예: 시험40%+진도40%+과제20%)
8. 운영 옵션: 수강료(무료/유료/할인), 공개 대상(지정회원/비공개), **동영상 건너뛰기 설정**, **사전 모집기간(얼리버드)**

### AI SAM (디디쌤 AI 어시스턴트)

- 교육 목표·주요 내용 자동 작성
- 클래스명 기반 썸네일 이미지 추천
- 클래스명+교육 대상자 → 커리큘럼 자동 생성

## 터치클래스 - 온라인 강좌(코스) 등록 플로우

1. **콘텐츠 선택** (개설 후 변경 불가) — 콘텐츠 페이지 기반
2. 코스 분류/명/소개/상세 상단 이미지(최대 5장)
3. 수강 등록 방식 — 회원 신청 OR 관리자 등록(그룹 자동)
4. 수강 기간 — 시작~종료일 / 시작일+N일 / 제한 없음
5. **수강 시간 제한** (특정 요일·시간만)
6. 코스 상세 정보:
   - **자리비움 모니터링** (학습 활동 없으면 학습시간 정지)
   - 커리큘럼별 공개일 설정
   - 페이지 순서대로 수강
   - 필수 코스 지정: **진도율 / 학습시간 / 시험점수 / 설문제출 / 동영상 학습조건**
   - 동영상 skip 제한, 재생률 기준

### 숏클래스 (터치클래스 AI)

- 콘텐츠 영상 → 10~60초 짧은 영상 자동 생성 (마케팅용 미리보기)
- 강좌 기획 AI는 없음

## 핵심 비교 (온라인 한정)

| 항목 | 디디쌤 | 터치클래스 |
|---|---|---|
| AI 강좌 기획 | **AI SAM (목표·내용·썸네일·커리큘럼)** | 없음 (영상 변환만) |
| 수료기준 설정 | **드래그 비율(40/40/20 식)** | 충족/미충족 이진 |
| 사전 모집·마케팅 | **사전 모집기간 + 얼리버드 할인** | 신청기간만 |
| 결제·구독 | 유료/무료/할인 + **구독 클래스** | 약함(B2B LMS) |
| 코스 묶음 운영 | **과정 클래스** | 없음 |
| 콘텐츠 변경 | 자유 | **개설 후 변경 불가** |
| 학습 측정 정밀도 | 진도율/시험/과제/설문 | **페이지별 학습시간 / 자리비움 / 재생률 / PDF 조회수** |
| 권한 분리 | 가이드에 명시 없음 | **부관리자 코스 단위 권한** |
| 수강 시간 제한 | 없음 | **요일·시간 제한** |
| 대량 등록 | 일괄 초대 | **CSV 대량 + 그룹 자동** |

## newcourse 적용안 (팀 합의)

가져갈 것 (온라인 강좌 빌더):
1. **클래스 유형 분기**(콘텐츠/오프라인/블렌디드) — 디디쌤 4분류 패턴
2. **AI 어시스턴트** (커리큘럼·썸네일·교육목표 자동 생성) — Phase 1 외 후순위
3. **사전 모집기간 + 얼리버드 할인**
4. **과정 클래스(코스 묶음)** 운영
5. **수료기준 드래그 비율 UI** — EnrollmentPolicy 후속 페이즈 레퍼런스
6. **개인별 수강 기간(N일 정책)** — newcourse 결정과 일치 (이미 반영됨)

기본기로 깔 것 (터치클래스 수준):
- 페이지·아이템별 진도율
- 체류시간(학습시간) 측정
- 영상 재생률 + skip 제한
- CSV 대량 수강생 등록

다음 검토:
- 오프라인 코스 비교는 디디쌤 **회차 클래스(요일반/특강반/회차 차감반) + 출결패드**를 별도 세션으로 정리.
