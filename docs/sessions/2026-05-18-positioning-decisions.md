## 2. spec v0.2의 핵심 변경 (vs v0.1)

### 새로 추가된 섹션
- **S0. 제품 포지셔닝** — 브랜드 구조 / 고객 분류 가설 / USP
- **S2. 3-layer 분리** — Presentation / Evidence / Fulfillment
- **S2.2 Evidence Store** — 4단계 데이터 파이프라인
- **S3.1 EnrollmentPlan** — 학습자별 동결 (5번째 객체)
- **S8. 코스 유형 USP** — newcourse가 비로소 가능하게 하는 것
- **S10. 운영 정책 방어선** — 정합성 2층 방어
- **S14. 레거시 관계** — 마이그레이션 미지원 명시

### 축소·재구성
- **마이그레이션** → "미지원, 향후 별도" 한 섹션으로 축소
- **오프라인 흡수** → S8 코스 유형의 한 종류로 재구성

### 표지 변경
- **한 줄 룰** 재작성 — 동등 옵션 + 그린필드 + 공존 없음 명시
- **이유 5개** 재작성 — 비즈니스/가치/타깃/리스크/그린필드

## 6. 한 줄 보고

> **newcourse는 터치클래스 브랜드 우산 아래 레거시와 동등한 옵션으로 운영되며, "에디터 없는 깔끔한 운영 + 새 코스 유형(오프라인·블렌디드·단독 에셋)이 비로소 가능"한 그린필드 시스템이다. 두 시스템은 코스·데이터 수준에서 공존하지 않으며, 마이그레이션은 현재 미지원이다. spec v0.2는 본 결정을 반영해 작성 완료.**

> ※ "작성 완료"는 문서 작성 행위의 종료를 의미하며, 본 spec의 "Phase 1 완료"를 의미하지 않는다. spec은 v0.2 draft 상태이며, 미해결 논의 8건의 결정에 따라 v0.3로 발전한다.
