1

수강 모드 비교

코스 상세

코스 상세 화면은 운영자가 설정한 수강 방식에 따라 3가지 형태로 렌더링됩니다. 화면 우하단 배지와 버튼 텍스트가 모드를 구분합니다.

자율 단일

학습자가 커리큘럼 1개를 직접 선택. 선택 전까지 신청 버튼 비활성.

자율 다중 조합

N개 조합 선택형. 설정 개수를 채워야 신청 버튼 활성화.

관리자 강제배정

운영자가 지정한 일정이 자동 선택된 상태로 노출. 모달 없이 바로 확인 화면으로 이동.

2

자율 단일 선택 흐름

single
①
"수강 신청하기" 버튼 → 일정 선택 모달 오픈

커리큘럼 목록이 바텀시트로 슬라이드 업. 마감·잔여석 상태가 배지로 표시됨.

모달 오픈
일정 선택 후
②
"신청 확인" 버튼 → 신청 확인 화면

선택한 일정 요약 카드 + 개인정보 수집 동의 + 최종 신청 버튼.

3

자율 다중 조합 흐름

multi
①
모달에서 N개 복수 선택

체크박스 방식. 설정 개수(예: 2개) 충족 시 "신청 확인" 버튼 활성화.

②
신청 확인 화면 — 항목별 상세 카드

단일 선택과 달리 선택한 각 커리큘럼의 날짜·시간·장소가 개별 카드로 펼쳐짐.

4

관리자 강제배정 흐름

assigned
코스 상세 (배정됨)
신청 확인

운영자가 지정한 일정이 자동 선택된 상태. 학습자는 모달 없이 "수강 신청" 버튼만 누르면 곧바로 신청 확인 화면으로 이동.

5

날짜선택형 회차 선택

sessionSelect

법정의무교육 등 특정 코스에서 적용. 커리큘럼 카드를 선택하면 내부에 회차 날짜 목록이 펼쳐지고, 원하는 날짜를 골라야 신청 확인으로 이동할 수 있음.

회차 날짜 선택 UI
신청 확인 (선택된 회차 날짜)
6

내 수강 내역 탭

screen-my

신청 완료 후 접근. 수강 신청 상태·출석 현황·취소 버튼을 한 화면에서 확인.

화면 흐름 요약
코스 상세 → 일정 선택 모달 (single/multi) → 신청 확인 → 완료 모달 → 내 수강 내역
assigned 모드는 모달 단계를 건너뜁니다.
