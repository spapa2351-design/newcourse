# 온라인 코스 등록 빌더 — PDCA 완료 리포트

---

## Executive Summary

| 관점 | 내용 |
|------|------|
| **Problem** | HRD 담당자가 온라인 코스 개설 시 기본정보·커리큘럼·수료조건·수강기간을 분산된 화면에 수동 입력. 콘텐츠 이수조건 연결 오류 발생 |
| **Solution** | 단일 화면 4섹션 폼 + 우측 커리큘럼 사이드바(챕터→항목 DnD). 라이브러리에서 콘텐츠 검색·이수조건 즉시 설정. 퍼블리싱 전 6개 체크리스트 자동 검증 |
| **Value Delivered** | 3개 갭 식별·전건 해결, Match Rate 84% → 100%. selRule(단일/다중) + changePolicy + 강제할당 안내 UI 완성 |
| **Core Value** | offline-builder와 함께 LMS 코스 등록 온라인 축 완결 — "코스 기획에서 퍼블리싱까지 한 화면에서" |
