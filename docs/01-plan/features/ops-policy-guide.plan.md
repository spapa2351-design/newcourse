# 운영팀 정책 가이드 — 공식 기획 Plan

---

## Executive Summary

| 관점 | 내용 |
|------|------|
| **Problem** | 오프라인 코스 운영 정책(출석 기준·수료 조건·수강신청 처리 등)이 개발자 중심 기획서(offline-spec.html)에만 있어 운영팀(카리나·태서)이 일상 업무 중 빠르게 참조하기 어려움 |
| **Solution** | 운영팀 전용 정책 가이드 HTML — 케이스별 처리 기준을 밀도 높은 표(Excel 스타일)로 정리하고 guide-offline-lms.html과 동일한 좌측 네비게이션 구조로 제공 |
| **기능/UX 효과** | "이 케이스 어떻게 처리해?" 질문이 생길 때 즉시 펼쳐서 참조할 수 있는 단일 HTML 파일. 케이스-상태-결과가 한 줄에 보이는 고밀도 테이블로 비개발자도 빠르게 스캔 가능 |
| **Core Value** | "슬랙 질문 없이 혼자 판단할 수 있는 운영 기준서" — 신규 운영자 온보딩과 일일 CS 응대의 1차 참조 문서 |

---

## Context Anchor

| 항목 | 내용 |
|------|------|
| **WHY** | 운영팀이 정책 판단을 매번 기획자에게 물어봐야 하는 의존 구조 해소 |
| **WHO** | 운영팀 (카리나, 태서) — 비개발자, 일상적 CS·출석·수료 처리 담당 |
| **RISK** | 기획서 내용과 가이드 내용이 동기화되지 않으면 혼선 발생. 정책 변경 시 두 문서를 동시에 업데이트해야 함 |
| **SUCCESS** | 운영팀이 슬랙 문의 없이 가이드만 보고 처리 결정 가능한 수준 |
| **SCOPE** | 단일 HTML 파일 (docs/guide-ops-policy.html). 오프라인 코스 Phase 1 정책만 포함 |

## 1. 콘텐츠 범위 (7개 챕터)

| # | 챕터명 | 핵심 내용 | 표 스타일 |
|---|--------|-----------|-----------|
| 1 | **출석 처리 기준** | 출석 상태 5종 정의, 상태별 트리거 조건, 지각 기준 10분, 수동 변경 가능 여부 | 고밀도 상태 매트릭스 |
| 2 | **회차 마감 절차** | 마감 전 체크리스트, 미확인→결석 일괄 전환, 마감 후 수정 불가 범위 | 순서형 체크리스트 + 표 |
| 3 | **수료 처리 기준** | 수료 조건(출석률 75%), 수료 확정 절차, 미수료 판정 케이스 | 조건 매트릭스 |
| 4 | **수강신청 관리** | 자율 신청 기간, 강제 할당, 정원 초과 처리, 중복 신청 차단 규칙 | 케이스-처리 2열 표 |
| 5 | **데이터 정정** | 출석 상태 오류 수정 절차, 정정 가능 기간, 요청→승인 플로우 | 플로우 + 조건 표 |
| 6 | **QR 현장 운영** | QR 코드 생성 방법, 30초 자동 갱신, 라이브 모드 ON/OFF, 스캔 실패 대응 | 단계별 절차 표 |
| 7 | **CS 응대 가이드** | 수강생 문의 유형 10종, 담당자 처리 방법, 기획팀 에스컬레이션 기준 | 문의유형-처리방법 2열 표 |

## 2. UI/UX 방향

### 레이아웃
- **기반**: `guide-offline-lms.html`과 동일한 좌측 네비게이션 + 메인 콘텐츠 구조
- **상단 바**: "운영팀 정책 가이드" 로고, 버전 배지
- **사이드바**: 7개 챕터 + 서브 섹션 네비게이션

### 정책 표 스타일 (Excel-like)
- **컬럼 헤더**: 배경 `bg-slate-100`, 굵은 글씨, 셀 border 명시
- **셀 경계**: `border border-slate-200` 모든 셀 — 엑셀과 동일하게 격자 표시
- **조건 셀**: 상태/케이스에 색상 배지(badge) 적용
- **행 hover**: `hover:bg-blue-50` 로 현재 참조 중인 행 시각화
- **고밀도**: `text-[12.5px]`, `py-1.5 px-2.5` — 많은 케이스를 한 화면에 표시

### 운영팀 친화 요소
- **주의 callout**: 빨간색 callout로 "이 케이스는 절대 하면 안 되는 것" 강조
- **TIP callout**: 초록색 callout로 현장 팁 제공
- **단축 색인**: 챕터 상단에 "이 챕터에서 답하는 질문들" 목록
