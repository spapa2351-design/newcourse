# Plan: course-delete-policy
> Feature ID: course-delete-policy  
> Created: 2026-04-28  
> Phase: 1 (MVP)
> Reference: 현재 서비스 삭제 다이얼로그 패턴 참고

---

## Executive Summary

| 관점 | 내용 |
|------|------|
| **Problem** | 코스 목록에서 단순 confirm으로 삭제가 가능해 운영자가 삭제 시 어떤 데이터가 사라지는지 인지하지 못한 채 실행하는 위험이 있다. |
| **Solution** | "삭제" 텍스트 입력 + 삭제 항목 목록 명시 + 수강생 수 경고를 포함한 2단계 확인 다이얼로그로 대체한다. 보관(Archive) 기능은 제공하지 않는다. |
| **기능 UX 효과** | 운영자가 삭제 전 데이터 영향 범위를 인지하고 의식적으로 확인 입력 → 실수 삭제 최소화. 현재 서비스 패턴과 일관된 UX 유지. |
| **Core Value** | 삭제는 단순하게, 하지만 충분한 정보를 주고 결정하게. |

---

## Context Anchor

### WHY (배경)
현재 서비스(참고 레퍼런스)는 보관 없이 완전 삭제만 제공하되, "삭제되는 것 / 유지되는 것" 목록 + 텍스트 입력 확인으로 안전 장치를 둔다. 오프라인 LMS도 동일 패턴을 따르되, 오프라인 특화 항목(차수, 회차, 출석 기록)과 수강생 수 경고를 추가한다.

### WHO (대상 사용자)
- **Primary**: LMS 관리자·운영 담당자 — 코스 삭제 실행
- **Secondary**: 없음 (수강생은 삭제 UI 접근 불가)

### RISK (위험 요소)
| 위험 | 완화 방안 |
|------|----------|
| 수강생 있는 코스 실수 삭제 | 다이얼로그에 수강생 N명 경고 + "삭제" 입력 확인 |
| 삭제 항목 범위 오인 | 삭제 목록 bullet 명시로 사전 고지 |
| 텍스트 복붙 통한 무심코 확인 | "삭제" 6자는 짧아 복붙과 직접 입력 차이 없음 — 현재 서비스와 동일 패턴 유지 |

### SUCCESS (성공 기준)
- [ ] Trash 아이콘 클릭 → 삭제 다이얼로그 열림
- [ ] 다이얼로그: 코스명, 삭제 항목 목록, 수강생 수(> 0일 때 경고), "삭제" 입력창 표시
- [ ] "삭제" 입력 일치 시 확인 버튼 활성화, 불일치 시 비활성
- [ ] 확인 클릭 → 코스 목록에서 즉시 제거
- [ ] 일괄 선택 삭제 시 동일 다이얼로그 패턴 적용 (코스 수 N개 표기)

### SCOPE (범위)
- **In-Scope**: 단건 삭제 다이얼로그, 일괄 삭제 다이얼로그, 수강생 수 경고, 삭제 항목 목록
- **Out-of-Scope**: 보관(Archive) 기능 없음, 복원 기능 없음, 권한별 분기(Phase 2 — 백엔드 연동 시), 삭제 감사 로그(Phase 2)

## Phase 1 범위 (MVP)

| # | 기능 | 설명 |
|---|------|------|
| F-01 | 삭제 다이얼로그 | Trash 아이콘 클릭 → overlay modal. 제목 "정말 삭제하시겠습니까?", 코스명 하이라이트 박스, 삭제 항목 bullet, "삭제" 입력창, 취소·확인 버튼 |
| F-02 | 수강생 수 경고 | 해당 코스 수강생 > 0일 때 경고 배지: "⚠️ N명의 수강생 데이터가 함께 삭제됩니다" |
| F-03 | 삭제 항목 목록 | bullet로 표시: 수강생 등록 정보, 차수·회차 일정/정원/장소, 회차별 출석 기록, 코스 통계 |
| F-04 | 텍스트 입력 확인 | 입력창 placeholder "삭제", value === '삭제' 일 때만 확인 버튼 활성화 (bg-red-600) |
| F-05 | 일괄 삭제 | selectedIds > 0 → 동일 다이얼로그, 코스명 대신 "선택한 코스 N개" 표기, 수강생 총합 경고 |

## 삭제 다이얼로그 UI 스펙

```
┌─────────────────────────────────┐
│  🔴 (체크 아이콘 / 경고 아이콘)     │
│                                 │
│  정말 삭제하시겠습니까?              │
│  선택한 코스와 관련된 모든 데이터가  │
│  삭제되며, 삭제 후에는 복구할 수    │
│  없습니다.                        │
│                                 │
│  🔔 삭제할 코스: [코스명]           │  ← 핑크 배경 박스
│                                 │
│  ⚠️ 23명의 수강생 데이터가          │  ← 수강생 > 0 일 때만 노출
│     함께 삭제됩니다.               │
│                                 │
│  · 수강생 등록 정보                │
│    (수강생 목록, 출석 기록 등)      │
│  · 차수·회차 일정/정원/장소         │
│  · 코스 통계                      │
│                                 │
│  계속하려면 삭제를 입력하세요.       │
│  [삭제________________]          │
│                                 │
│  [    취소    ]  [  확인  ]       │
└─────────────────────────────────┘
```

확인 버튼: 기본 `bg-slate-200 text-slate-400` → 입력 일치 시 `bg-red-600 text-white`
