온라인 코스

온라인 코스는 newcourse 시스템의 콘텐츠형(CONTENT) 코스 유형입니다. 영상·시험·설문·PDF를 슬롯 단위로 묶어 진도율 100% 정합을 보장하며, 의무 교육·법정 교육·표준 직무 강의에 적합합니다.

v1.0 · 2026-05-21 코스v2 그룹 · 온라인 코스 코드: CONTENT

00 / Summary한눈에 보기

한 줄 룰

코스 = 커리큘럼(그룹) → 슬롯(에셋 + 수료 조건)의 순서 인덱싱. 모든 측정·이수·자격은 슬롯 단위로 정밀하게 한다.

다섯 가지 이유

정합성 — 페이지 머무르기로 진도율 부정 학습 차단.

슬롯 단위로 측정하면 영상은 실제 재생 시간, 시험은 점수 기준. 학습 안 하고 페이지 열어두는 행위가 통하지 않는다.

재사용성 — 같은 슬롯이 컴플라이언스·부트캠프·자유학습 케이스 모두 표현 가능.

슬롯 단위 기능이 다양한 코스 유형을 동일한 모델로 지원한다.

운영자 부담 최소화 — 단순 케이스는 일괄설정, 복잡 케이스는 슬롯별 정밀 편집.

"전체 순서대로 수강", "모두 필수" 같은 일괄설정으로 빠르게 만들고, 필요하면 슬롯별로 세부 편집.

확장성 — 블렌디드(Phase 2)도 같은 모델 재사용.

온라인 슬롯 + 오프라인 슬롯이 같은 코스 안에 공존 가능.

콘텐츠 재사용 — 라이브러리에서 한 번 올린 영상·PDF를 여러 코스에 참조.

중복 업로드 없음. 사용처가 어디인지 추적 가능.

01 / Builder코스 빌더

단일 화면 4섹션 폼 + 우측 커리큘럼 사이드바 + 좌측 진행도 미니맵.

화면 구조

헤더← 뒤로 / 코스명 / 상태칩 / 커리큘럼 버전 칩 / 임시저장 / 학습 뷰어 미리보기 / 퍼블리싱
좌측 미니맵등록 진행도 (N / M 필수) · 4 step (코스 기본 정보 · 수강 신청 설정 · 수강 기간 · 커리큘럼·항목)
섹션 1 코스 기본 정보코스명 / 커버 이미지(720×1200, 5MB) / 카테고리(optgroup) / 강사(default 강사 없음 ON) / 강사 소개 / 코스 소개
섹션 2 수강 신청 설정신청 기간 / 등록 방식(자율/강제 라디오 — 동시 ON X) / L1 코스 등록 자격(그룹 픽커)
섹션 3 수강 기간 설정시작일~종료일 / 등록 후 N일 / 무제한
섹션 4 학습 진행 룰의무 교육: 수강 시간 제한 · 자리비움 모니터링 · 서명 문서 / 일정 학습: 단계별 공개
우측 사이드바커리큘럼 관리: 커리큘럼 → 항목(슬롯) 2계층, DnD, 라이브러리 가져오기, [⚙ 일괄설정] 모달

빌더 chrome-off

빌더는 풀스크린 작업 모드. 글로벌 사이드바·탑바 제거, 빌더 자체 헤더만 유지. 운영자가 한 번에 코스 구성에 집중.

일괄설정 모달

우측 사이드바 [⚙ 일괄설정] 버튼이 모달 트리거. 모달 안에 3섹션:

① 필수/선택모두 필수 / 모두 선택 / 영상만 필수 — 3종 액션
② 순차 학습전체 순차 진입 적용 / 모두 해제 — 항목별 prerequisite 일괄 설정
③ 공개일 (단계별 공개)N일 간격 적용 + 커리큘럼별 개별 편집

액션 클릭 → 모달 하단 amber 패널에 변경 요약 → [적용] / [취소] 2단계.

퍼블리싱 검증 체크리스트

  • 코스명 입력
  • 담당 강사 지정 (또는 강사 없음 토글)
  • 신청 기간 설정
  • 수강 기간 설정
  • 커리큘럼 최소 1개
  • 필수 콘텐츠 최소 1개

모두 충족 시 [퍼블리싱] 버튼 활성. 미충족 시 미니맵에서 빨간 동그라미 표시.

구현 파일

admin/online-builder.html — React 18 CDN + Tailwind CDN + Lucide. 단일 HTML 파일, 백엔드 미연결 (mock).

02 / List코스 목록

레거시 목록 셸 차용. 1000+ 코스 페이지네이션 처리.

레이아웃

좌측 글로벌 사이드바(200px) + 좌측 분류 트리(180px) + 메인 카드 (상태 탭 + 필터 칩 + 검색 + 테이블 + 페이지네이션).

컬럼 (11개)

☐체크박스 (전체선택 헤더 포함)
상태5종 칩 — 초안 · 운영중 · 수강중 · 종료 · 관리자 종료
표지40×60 portrait 썸네일
코스명bold + prefix [필수] 칩 (필수 코스만) + 하단 등록 채널 미세 라벨
커리큘럼"커리큘럼 N · 에셋 M" 요약 칩
수강관리[수강관리] 버튼 (자체 컬럼)
분류분류명 칩
수강 기간"제한 없음" 또는 날짜 범위
수강 인원N명
등록일날짜 + 시간
관리링크·편집·삭제 3아이콘

페이지네이션

  • 디폴트 10개/페이지 (10/20/50/100 셀렉트)
  • 중앙 페이지 번호 + 좌우 화살표 + ellipsis (1·2·3 … 마지막)
  • 필터·검색·정렬 변경 시 1페이지 자동 리셋

상태 5종

초안draft. 퍼블리시 안 됨. 운영자에게만 보임. slate
운영중published + 신청 기간 진행 중. 신규 등록 가능. emerald
수강중신청 기간 끝. 수강만 진행. 신규 등록 불가. amber. (학습자 측 라벨은 "신청 마감")
종료수강 기간까지 끝. 학습 종료. slate dim
관리자 종료관리자가 강제 종료. 즉시 차단. rose

일괄 액션 (1차)

  • 일괄 삭제만 (운영중·수강 인원>0 가드)
  • 일괄 종료·공개·분류 변경은 v2

구현 파일

admin/online-list.html — 레거시 코스 목록 셸 복제 + newcourse 모델 정합.

03 / Library콘텐츠 라이브러리

영상·PDF를 한 곳에 모아 여러 코스에서 재사용.

한 줄 룰

콘텐츠는 라이브러리에 한 번만 존재하고, 코스는 라이브러리 항목을 참조한다.

핵심 동작 (1차)

콘텐츠 타입영상 · PDF (시험·설문은 별도 기능 후속)
등록 흐름2-way: 라이브러리 직접 업로드 + 빌더에서 만들 때 자동 적재
검색·필터텍스트 + 타입 칩(영상/PDF) + 정렬(등록일·이름·사용 빈도)
사용처 추적카드에 "사용 코스 N개" 칩 + 클릭 시 사용처 리스트 모달
삭제 가드사용 중 콘텐츠 삭제 차단. 사용처 안내 다이얼로그
편집메타(이름·설명·썸네일)만 라이브러리에서. 본문은 빌더 안
권한단일 tenant 조직 공용 (멀티테넌트 마스터는 v2)

테이블 컬럼 (목록 뷰)

  • ☐ · 타입(영상/PDF) · 썸네일(40×24 영상 / 24×24 PDF) · 제목+설명
  • 메타(재생시간/페이지수) · 용량(MB) · 업로드일 · 등록자 · 사용 코스 · 액션

상세 모달

  • 영상: 인라인 <video controls> 플레이어
  • PDF: 인라인 <iframe> 뷰어 (480px 높이)
  • 메타 인라인 편집 (제목·설명)
  • 사용 코스 리스트 (각 코스로 이동 링크)
  • 푸터: [삭제] · [↓ 다운로드] · [닫기]

구현 파일

admin/content-library.html — 사이드바 신규 영역에서 진입.

04 / Model데이터 모델

코스·커리큘럼·슬롯의 위계와 핵심 필드.

Course (코스)

Course {
  id, title, coverImage,
  source: 'v2',
  type: 'CONTENT',
  classification: 'normal' | 'mandatory',
  category, status,
  enrollMode: 'admin' | 'self' | 'auto',
  studyPeriod: { type: 'fixed' | 'days' | 'unlimited', start?, end?, days? },
  applyPeriod: { start?, end? },
  enrolledCount, createdAt, createdBy,
}

Curriculum (커리큘럼)

Curriculum {
  id, title,
  releaseAfter: number,    // 단계별 공개 일수
  slots: Slot[]
}

Slot (슬롯 = 에셋 + 수료 조건)

Slot {
  id, type: 'video' | 'exam' | 'survey' | 'pdf',
  title, isRequired,
  cond: { /* type별 */ },
  source: 'library' | 'upload' | 'empty',
  libRef,
  prerequisiteIds: slotId[]
}

슬롯 타입별 수료 조건

videowatchRate (시청률 %) — 기본 90
exampassScore, totalQ — 기본 60점 / 20문항
survey(없음 — 제출 기준)
pdf(없음 — 조회 기준)

06 / RoadmapPhase 로드맵

PHASE 1
현재
빌더·목록·라이브러리 + 슬롯 단위 진도율

코스 등록·수정·삭제 + 슬롯 4종(video·exam·survey·pdf) + 라이브러리 영상·PDF + 일괄설정 모달.

  • 빌더 4섹션 폼 + 우측 사이드바 + chrome-off
  • 목록 11컬럼 테이블 + 페이지네이션 + 일괄 삭제
  • 라이브러리 영상·PDF + 사용처 추적 + 삭제 가드
  • 일괄설정 모달 (필수/선택 · 순차 학습 · 단계별 공개)
PHASE 2
확장 정책 시점 미정
컴플라이언스 옵션 · 단계별 공개 정착

의무 교육 운영자 요구를 위한 시간대 제한·자리비움 모니터링·서명 문서 실 동작.

  • 수강 시간 제한 (요일별 시간 윈도우)
  • 자리비움 모니터링 (N분 inactive)
  • 서명 문서 첨부 (감사 대비)
  • 커리큘럼 버전 발행 로직
PHASE 3
멀티테넌트 라이브러리 시점 미정
슈퍼관리자 마스터 라이브러리 + 레퍼런스/Fork

법정 의무 영상 같은 표준 콘텐츠를 슈퍼관리자가 N개 고객사에 배포.

  • Master Library (슈퍼관리자 전용)
  • 레퍼런스 디폴트 + 명시적 fork
  • master 업데이트 시 freeze 룰 (학습 중 학습자 보호)
  • 회수(라이센스 만료) · grace period
